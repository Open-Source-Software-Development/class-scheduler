default ImmutablePair<L, R> osd.util.pair.ImmutablePair< L, R >.immutable ( )
inline

Get an immutable copy of this pair. If this pair is already immutable, return the pair itself. Otherwise, construct a new immutable pair with the same elements. If the pair was mutable, changes will not be reflected in the immutable copy.
