---How To Run Test---
1. Download/Install Google Chrome
2. Navigate to the Chrome App Store
3. Install Katalon Automatic Recorder
4. Open Katalon Automatic Recorder through Chrome Window
5. On the left hand side next to "Test Suites" view click folder icon "Open Test Suites"
6. Select the .html files
7. In the UPLOAD tests modify the location of the test file directory on local pc
